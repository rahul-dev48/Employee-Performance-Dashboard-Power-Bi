📊 Employee Performance Dashboard

A complete end-to-end Data Analytics Project that analyzes employee performance using Python, SQL, and Power BI. The project demonstrates the full analytics workflow—from data cleaning and exploratory data analysis (EDA) to SQL querying, dashboard creation, report generation, and presentation.

---

📌 Overview

The Employee Performance Dashboard helps organizations understand employee productivity, performance trends, salary distribution, attendance, and promotion insights through interactive visualizations and data-driven analysis.

This project showcases practical skills in:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Querying
- Data Visualization
- Business Intelligence
- Report Creation
- Presentation Design

---

📂 Dataset

The dataset contains employee-related information, including:

- Employee ID
- Employee Name
- Department
- Job Role
- Gender
- Age
- Experience
- Salary
- Performance Rating
- Attendance
- Training Hours
- Promotion Status

The dataset was cleaned and transformed before analysis to ensure accuracy and consistency.

🔄 Project Workflow

1. Data Loading

- Imported the dataset into Python.
- Verified column names and data types.
- Explored dataset structure.

2. Data Cleaning

- Removed duplicate records.
- Handled missing values.
- Corrected inconsistent data.
- Converted data types.
- Prepared the dataset for analysis.

3. Exploratory Data Analysis (EDA)

Performed detailed analysis to discover meaningful insights.

Examples:

- Department-wise employee performance
- Salary distribution
- Performance rating analysis
- Attendance trends
- Experience vs Performance
- Promotion analysis

Visualizations were created using Matplotlib and Seaborn.

4. SQL Analysis

Business insights were generated using SQL queries in PostgreSQL/MySQL/SQL Server.

Examples:

- Top-performing employees
- Department performance summary
- Average salary by department
- Promotion statistics
- Employee count by department
- Attendance analysis

5. Power BI Dashboard

An interactive dashboard was created to visualize key performance metrics and business insights.

6. Report Creation

A professional report summarizing findings, KPIs, and recommendations was created.

7. Presentation

A presentation was designed using Gamma AI to communicate project outcomes effectively.

▶️ How to Run

Step 1

Clone the repository.

git clone <repository-link>

Step 2

Install the required libraries.

pip install -r requirements.txt

Step 3

Open the Jupyter Notebook.

jupyter notebook

Step 4

Run the Python notebook to perform:

- Data Loading
- Data Cleaning
- EDA

Step 5

Execute SQL queries using:

- PostgreSQL
- MySQL
- SQL Server

Step 6

Open the Power BI (.pbix) file to explore the dashboard.

Step 7

Review the final report and presentation.
